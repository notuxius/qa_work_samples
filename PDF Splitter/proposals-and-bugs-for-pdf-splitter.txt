Общее:
1. У программы слабая локализация.
2. Во время выполнения долгих операций - программа становится менее отзывчивой - добавить возможность менять приоритет процесса программы для выполнения операций и/или делать его фоном.
3. Во время выполнения операций в окне "Конвертация" возле поля "Итог" присутствуют две единицы - их назначение не ясно.

Главное окно:
1. Программа зависает при выборе многих файлов.
2. Спиннер программы можно переделать в маленький спиннер - и разместить его возле текста "Question? We offer you a..." внизу интерфейса - и мгновенно отображать его при каждой операции - это придаст пользователю ощущение большей скорости отклика интерфейса программы.
3. В интерфейсе можно разделить список обычных файлов и файлов полученных из программы.
4. Возле кнопки "Включая поддиректории" можно добавить кнопку "Показывать все файлы" - вместо надписи "<некоторые файлы отфильтрованы...>"
5. Выпадающее меню "Фильтр" может переключаться в режим "All files (*.*)" при клике на надписи "<некоторые файлы отфильтрованы...>".
6. Возможность включить или отключить виджеты списка директорий и предварительного просмотра - например кнопками на панели инструментов и в меню.
7. Переместить строку поиска из нижней части интерфейса - в правую верхнюю часть - традиционное размещение строки поиска - там где чаще всего его ожидает увидеть пользователь.
8. При изменении слайдера масштаба внизу - есть смысл менять размер шрифта для всего интерфейса - а не только в выборе директорий и в списке файлов - возможно добавить надпись "Размер шрифта" возле слайдера слева и отображать проценты справа слайдера - добавить возможность изменять масштаб колёсиком мышки на слайдере.
9. Растягивать столбцы таблицы на всё окно списка выбора файлов - для максимально эффективного использования пространства интерфейса при минимальном масштабе шрифта.
10. Выпадающее меню "Фильтр" можно уменьшить по ширине - кнопку "Перейти" можно переместить вниз и справа от меню "Фильтр" - кнопку можно переделать в поле ввода "Перейти в папку" - для перехода в нужную папку без необходимости нажимать на неё - так как у кнопки только одна функция и после перемещения вниз будет место для размещения поля ввода "Перейти в папку".
11. Убрать возможность добавлять в избранное одной и той же папки несколько раз.
12. Возможно переместить панель "Включая поддиректории Отменить Снять..." ниже на панель возле изменения масштаба - для увеличения полезного пространства.
13. Добавить возможности удалять и переименовывать файлы прямо в интерфейсе программы.
14. В меню правого клика мышкой на файлах - добавить функции из панели "Split by".
15. Добавить иконки для всех элементов интерфейса - например для меню "Файл - Обновить" "Файл - Options" "Файл - Выход" - также для кнопок "Включая поддиректории" "Отметить" "Снять"... - для более быстрой визуальной навигации по интерфейсу.

Виджет предварительного просмотра:
1. Возможность менять масштаб в предварительном просмотре нажатием клавиши Ctrl и колёсиком мышки.
2. Возможность перетягиванием перемещаться по видимой области файла.
3. Использовать два ряда вертикальных точек на панели для изменения ширины предварительного просмотра - также как на панели для изменения ширины списка директорий - для большей последовательности интерфейса.
4. Кнопки "Увеличить" "Уменьшить"... меняются местами после кликания на них - не знаю баг это или фича.
5. Кнопки "Увеличить" "Уменьшить"... можно сделать меньшего размера - для размещения их всех на панели - без необходимости дополнительно кликать на выпадающее меню с двумя стрелками для выполнения операций с масштабом файла - рассчитать возможность размещения всех кнопок на панели для стандартного файла А4.
6. Иконки кнопок "Увеличить" "Уменьшить"... не подходят по стилю иконкам на панели "Split by" - также по стилю не подходят иконки в окне "Manual split".
7. На панель к кнопкам "Увеличить" "Уменьшить"... можно добавить кнопки для навигации по следующим и предыдущим страницам.

Окно "Advanced filter":
1. В поля "More than" и "Less than" добавить стрелки вверх и вниз для увеличения и уменьшения размеров посредством кликов на стрелках, также добавить выпадающее меню с выбором размеров в килобайтах, мегабайтах и гигабайтах. 
2. Возможность ввести даты вручную - в поле "Newer than" - отображать дату на месяц назад от текущей, в поле "Older than" - отображать текущую дату.

Окно "Отчет":
1. Для отчётов можно использовать нестандартное окно завершения процесса - после составления отчёта - можно не отображать его автоматически - в окне "Process complete" кнопку "View log" можно переименовать в "Отобразить отчёт" и она будет служить для отображения отчёта - кнопку "Сохранить как..." можно убрать.
2. Возможно для всех типов отчётов кроме Excel и HTML (для CSV PDF и текстовых файлов) - использовать внутренний просмотрщик программы - а не внешний - так как внешнего может и не быть.
3. Для файлов отчётов вместо имён "Отчет" "Отчет (новый)" "Отчет (новый) (новый)" - использовать отметку времени - "Отчет-11.03.2015-12.45.00" - тоже самое и для логов.
